תיעוד ה-Rest API של השרת

ישות (Resource) שיטה (Method) נתיב (Endpoint) תיאור הרשאות
MakeUp Products GET /api/product שליפת כל המוצרים (כולל Paging - ?page=1&limit=10) כולם
MakeUp Products GET /api/product/:id שליפת מוצר לפי קוד מוצר כולם
MakeUp Products POST /api/product הוספת מוצר חדש מנהל בלבד
MakeUp Products PUT /api/product/:id עדכון מוצר מנהל בלבד
MakeUp Products DELETE /api/product/:id מחיקת מוצר מנהל בלבד
Users POST /api/user רישום משתמש חדש (מחזיר Token) כולם
Users POST /api/user/login כניסת משתמש (מחזיר Token) כולם
Users GET /api/user שליפת כל המשתמשים מנהל בלבד
Orders GET /api/order שליפת הזמנות של המשתמש המחובר משתמש מחובר
Orders GET /api/order/all שליפת כל ההזמנות מנהל בלבד
Orders POST /api/order יצירת הזמנה חדשה משתמש מחובר
Orders PUT /api/order/:id עדכון סטטוס "יצאה לדרך" בעלים של ההזמנה או מנהל
Orders DELETE /api/order/:id מחיקת הזמנה (רק אם לא נשלחה) מנהל בלבד
